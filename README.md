# go-scaffold
golang microservice scaffold


## Dependencies
- Web: iris
- Swagger: swagger
- ORM: gorm
- S3: minio
- Database: mysql
- Redis: go-redis
- Config: viper
- Logger: logrus
- Validator: go-playground/validator
- Test: testify
- Mock: mockery
- Docker: docker-compose
- CI/CD: github actions
